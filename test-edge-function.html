🧪 Test PriceCharting Image Scraper

This tool helps you test the edge function directly and see what's happening.

📋 Manual Deployment Instructions

Since the Supabase CLI isn't available, you need to deploy the function manually:

  1. Go to your Supabase Dashboard
  2. Navigate to Edge Functions
  3. Click on the price-charting-images function
  4. Replace the code with the updated version from supabase/functions/price-charting-images/index.ts
  5. Click Deploy
